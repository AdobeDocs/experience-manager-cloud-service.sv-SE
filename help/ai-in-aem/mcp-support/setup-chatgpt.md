# Konfigurera OpenAI ChatGPT med AEM MCP {#setup-chatgpt}

Följ de här stegen för att ansluta OpenAI ChatGPT till AEM MCP-servrar.

* Lägg till en eller flera AEM MCP-server-URL:er i området där MCP-anslutningar eller verktyg är konfigurerade.
* Utlös anslutningen och logga in med din Adobe ID när den omdirigeras.
* I en chatt refererar du till de konfigurerade AEM-verktygen i dina uppmaningar, till exempel:

  ```
  "Using the configured AEM MCP tools, list all sites in the author environment."
  ```
