# Konfigurera markör med AEM MCP {#setup-cursor}

Följ de här stegen för att ansluta pekaren till AEM MCP-servrar.

* I Markörens MCP-inställningar skapar du en ny MCP-serverpost med en eller flera AEM MCP-URL:er.
* Autentisera med din Adobe ID när du uppmanas till det.
* Du kan även aktivera eller inaktivera enskilda verktyg genom att klicka på verktygsnamnen. Alla verktyg är aktiverade som standard.
* Använd Markörens redigerare eller chatt för att starta AEM Tools som en del av arbetsflödena för utveckling och innehåll.
